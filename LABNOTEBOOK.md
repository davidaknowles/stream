- Benchmarked UCE as a replacement cell-state representation. The official 33-layer checkpoint loaded strictly into the reference architecture (1,280 hidden dimensions, 20 attention heads, 5,120-dimensional feed-forward layers). UCE samples 1,024 genes per cell. Atlas-derived tokenization produced only 1,063- and 1,065-token cells, so batches can be bucketed by exact length. At 1,065 tokens, checkpoint-backed BF16 inference with the reference padding mask sustained 116.8 cells/s (27.2 GPU-hours for all 11,441,407 cells). Bucketing and omitting the padding mask for exact-length batches selected PyTorch Flash Attention, sustained 131.2 cells/s, and projects to 24.2 GPU-hours on one GPU. The masked and mask-free BF16 embeddings had mean cosine similarity 0.999972. Batch sizes 50 and 100 had equivalent throughput; batch 50 is the practical initial setting.
- Sparse backed AnnData reads and UCE-style weighted gene sampling were measured at 487 cells/s (about 6.5 CPU-hours for the atlas), so they can be pipelined with GPU inference. The reference UCE preprocessor is unsuitable as-is because it materializes a dense `int64` cell-by-gene memmap; production embedding must stream sparse rows and write the 1,280-dimensional output incrementally. The final output alone is about 54.6 GiB as float32.
- Added the UCE-state STREAM ablation. UCE checkpoints and metrics use a `_uce` suffix so expression-state results are preserved.
